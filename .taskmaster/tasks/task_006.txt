# Task ID: 6
# Title: Create Open House Management EntryPoint
# Status: pending
# Dependencies: 1
# Priority: medium
# Description: Develop a custom EntryPoint to provide a public-facing digital sign-in form for open houses that creates leads in the CRM.
# Details:
Create a new EntryPoint in custom/EntryPoint/OpenHouseSignIn.php registered in custom/Extension/application/Ext/EntryPointRegistry/OpenHouse.php. Design a mobile-friendly HTML form using Bootstrap that collects visitor information (name, email, phone, notes). Create a secure token system to associate the form with a specific property listing. Implement server-side validation for all form inputs. When submitted, create a new Lead record with a relationship to the property. Set the lead_source to 'Open House'. Send an email notification to the agent when a new lead signs in. Create a QR code generator that agents can use to create a property-specific sign-in link. Implement rate limiting to prevent form spam. The URL format should be: index.php?entryPoint=OpenHouseSignIn&property_id=XXX&token=YYY.

# Test Strategy:
Test form submission with valid and invalid data. Verify leads are created correctly with all submitted information. Test the relationship between leads and properties. Verify email notifications are sent to agents. Test the form on various mobile devices and browsers. Test security measures against common attacks (CSRF, XSS, SQL injection).
