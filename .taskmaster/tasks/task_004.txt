# Task ID: 4
# Title: Implement Commission Calculator Logic Hook
# Status: pending
# Dependencies: 2
# Priority: medium
# Description: Create a before_save Logic Hook on the Opportunities module to automatically calculate agent commissions when a deal is marked as closed.
# Details:
Create a custom Logic Hook in custom/modules/Opportunities/logic_hooks.php that triggers on before_save. Implement the hook function to check if sales_stage is changing to 'Closed Won' (or the renamed equivalent). When triggered, calculate the commission based on the amount field and a configurable commission rate (stored in config or as a user preference). Store the result in the custom commission_c field. Add configuration options in the admin panel to set default commission rates (e.g., 2.5% or 3%). Include logic to handle split commissions if needed. The hook should follow this pseudocode: if (bean.sales_stage == 'Closed Won' && bean.sales_stage_changed) { bean.commission_c = bean.amount * (bean.commission_rate / 100); }. Use SuiteCRM's BeanFactory to ensure proper saving.

# Test Strategy:
Test that commission is correctly calculated when a transaction is moved to Closed Won. Verify that changing other fields doesn't trigger recalculation. Test with different commission rates and transaction amounts. Verify the commission appears correctly on the detail view. Test edge cases like zero amount or null commission rate.
