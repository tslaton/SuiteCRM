# Task ID: 7
# Title: Implement CMA Generator
# Status: pending
# Dependencies: 1, 3
# Priority: medium
# Description: Create a Comparative Market Analysis (CMA) generator that produces a client-ready PDF showing the subject property and comparable properties.
# Details:
Create a custom action in the Properties module (custom/modules/Properties/actions/GenerateCMA.php). Implement logic to query the mock_mls_data table for 3-5 comparable properties based on location, size, and features. Use TCPDF (already included in SuiteCRM) to generate a professional PDF report. Design a template for the PDF that includes: agent branding, subject property details, comparable property details with photos, price comparison chart, and market analysis summary. Add a custom button to the Property detail view to generate the CMA. Include configuration options for CMA parameters (e.g., search radius, price range percentage). Store generated CMAs as Documents linked to both the Property and related Contacts. Implement this pseudocode: function findComparables($property) { return $db->query("SELECT * FROM mock_mls_data WHERE zip = '{$property->zip}' AND bedrooms BETWEEN {$property->bedrooms-1} AND {$property->bedrooms+1} AND bathrooms BETWEEN {$property->bathrooms-1} AND {$property->bathrooms+1} AND square_footage BETWEEN {$property->square_footage*0.8} AND {$property->square_footage*1.2} LIMIT 5"); }

# Test Strategy:
Test CMA generation for various property types and locations. Verify the PDF contains all required sections and formatting is correct. Test with edge cases (no comparables found, missing data). Verify the PDF is correctly stored as a Document. Test performance with large property datasets. Verify the PDF renders correctly when viewed or printed.
