# Task ID: 10
# Title: Implement Transaction Pipeline Widget
# Status: pending
# Dependencies: 2, 5, 8
# Priority: medium
# Description: Create a widget for the Real Estate Hub that provides a summary view of the agent's transaction pipeline.
# Details:
Create a custom dashlet class (custom/modules/Home/Dashlets/PipelineDashlet/PipelineDashlet.php) that extends DashletGeneric. Implement a query to count transactions by stage and calculate total value for each stage. Create a Smarty template that displays a horizontal pipeline visualization with counts and values for each stage. Use Bootstrap progress bars or a simple chart library (Chart.js 2.9.x, compatible with SuiteCRM 7.x) to visualize the pipeline distribution. Include a summary of total pipeline value and count. Add a link to the full Kanban view of the pipeline. Implement a refresh button to update the data without reloading the page. Use color coding to highlight stages with high value or that require attention. The widget should be compact but informative, focusing on key metrics rather than individual transaction details.

# Test Strategy:
Test the widget with transactions in various stages. Verify counts and calculations are correct. Test the visualization with different distribution patterns. Verify the link to the Kanban view works correctly. Test the refresh functionality. Verify the widget updates when transactions change stages.
